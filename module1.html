What is HTML?

HTML= Hyper Text Markup Language

  • H=Hyper which means Reference/link
  • T=Text which means Data/Information
  • M= Markup Which Means Predefined/Marked
  • L= Language Which Means Communication

HTML Structure

<!DOCTYPE html >
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document </title>
</head>
<body>
</body>
</html>

HTML Structure Defination

  • Doctype:Gives information to the browser about the document type and version
  • html:Represents the root of an HTML document. It is a container for all the html elements except for the Doctype
  • Head:A container for metadata and is placed between the <html> tag and the <body> tag.
  • meta: defines metadata about an HTML document. Metadata is data(information) about data.
  • title: tag defines the title of the document.
  • body:contains all the contents of an HTML document
  • comment:a section of text that is not processed by the web browser. Comments are enclosed in <! ----> tags.

Block vs Inline

By default, some html tag are block element and some are inline element. Block elements are occupied total width of device. on the other hand, inline elements are occupied as the element's content needed. Block elements have width and height property, on the other hand inline elements don't have width and height property. To give inline element width and height property we use display: inline-block in css.

Html List(ordered, unordered)

Anchor Tag

The <a> tag defines a hyperlink, which is used to link from one page to another. Some attributes:

  • href: Specifies the URL of the page link gose to
    <a href="http://www.google.com"> Google </a>
  • target:Specifies where to open the linked document. by default linked page will load in the same tab but we can set to open linked page different tap by setting target = "_blank" property.
    <a href="http://www.google.com" target = "_blank" > Google </a>
  • download:Specifies that the target will be downloaded when a user clicks on the hyperlink
    <a href="./images/94225.png" download > Google </a>

Image Tag

The<img> tag is used to embed an image in an HTML page. Some attribute

  • src: Specifies the path to the image
    <img src="./image/logo.png">
  • alt:Specifies an alternate text for the image, if the image for some reason cannot be displayed
    <img src="./image/logo.png" alt ="This is a picture">
  • height:Specifies the height of an image.
    <img src="./image/logo.png" alt ="This is a picture" height="300px">
  • height:Specifies the width of an image.
    <img src="./image/logo.png" alt ="This is a picture" height="300px" width= "400px">

Image path

The image path is used to specifies of an image in an HTML page.

  • <img src="picture.jpg">
    Specifies that picture.jpg is located in the same folder as the current folder
  • <img src="images/picture.jpg">
    Specifies that picture.jpg is located in the images folder in the current folder
  • <img src="/images/picture.jpg">
    Specifies that picture.jpg is located in the images folder in the root of the current folder
  • <img src="../picture.jpg">
    Specifies that picture.jpg is located in the folder one level up from the current folder
