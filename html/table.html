Html Table

Table has comumn and row. First row is table head and rest is table body. in body, there have many row contains data.

table tags

  • Table: table
  • Table Head: thead
  • Table Body: tbody
  • Table Footer: tfoot
  • Table Row: tr
  • Table data or table cell: td
  • Table header cell: th
  • Table Caption: caption
