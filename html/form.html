HTML Form

The HTML <form> is used to create a form

Form and its attributes

A form is essentially a container for inputs

The following code shows the basic structure of a form

<form>
The input elements goes header
</form>

  • The action attribute points to the back-end of our web page, which handles the form submition
  • The method attribute is used to upload the data. The most commonly used attributes are the GET and POST method.

The Html form tag is required when you want to collect information that users provide.

For example. You may want to collect specific data from users, such as name, email address and passowrd.

The input tags

Input type Password

The <input> type="passowrd" defines a password field.

Input type submit

The <input> type="summit" defines a button for submitting form data to a server.

Input type radio

The <input> type="radio" defines a radio button. The radion button is used when only a single option needs to be selected out ot several offered options.

Input type checkbox

The <input> type="checkbox" defines a check button. The check button is used when multiple options needs to be selected out ot several offered options.

The <label> tag

The <label> tag represents a caption for an item in auser interface

The <textarea> tag

The <textarea> tag is used when the user has to enter details that may exceed a single sentence.

Select and option tag

  • The <select> element is most often used in a form, to collect user input
  • The name attribute is needed to reference the form data after the form is submited.
  • The id attribute is needed to associate the drop-down list with a label.
  • The <option> tags inside the <select> element define the available options in the dropdown list
